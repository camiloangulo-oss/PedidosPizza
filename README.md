# Sistema de Gestión de Pizzas con Pilas

## Objetivo
Implementar una pila manual usando listas ligadas en Java.

## Funcionalidades
- Registrar pizzas
- Undo
- Redo
- Mostrar pedido actual

## Tecnologías
- Java
- Listas Ligadas
- Pilas Manuales

## Ejecución
1. Compilar:
javac *.java

2. Ejecutar:
java GestionPedidos
